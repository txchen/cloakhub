# Preserve CloakBrowser Manager profile features

CloakHub v1 will support the profile features provided by CloakBrowser Manager that affect browser identity, persistence, and launch behavior, including fingerprint, proxy, locale/timezone, platform, screen, GPU, hardware concurrency, user agent, launch args, tags, notes, clipboard preference, and sleep-related settings. CloakHub's differentiator is resource-efficient Browser Instance lifecycle management, not reducing the core profile configuration surface.
